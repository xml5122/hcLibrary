向网页中添加flash影片
1、将flash文件转换成flv文件（搜索视频转换器）；
2、找一个可以播放此类型文件的播放器；
3、从浏览器下载引入flash的js，在head里引用；
4、在js里调用swfobject.embedSWF("文件位置","对应div的id","长","宽","支持flash的最低版本")

向网页中添加视频
video支持的格式，为了浏览器兼容至少准备H264和WebM两种格式；

video特性
src表示指定视频的路径；
poster表示在视频加载之前画面上显示的一张图片；
controls表示需要浏览器提供默认的播放器；
autoplay表示是否自动播放；
loop表示是否循环播放；
preload(none、auto、matadata)表示用户在点击播放按钮前是否自动加载视频 none表示不必、auto表示页面加载时加载、metadata表示浏览器只需要收集少量视频信息。

source:src表示视频路径；
type表示告诉浏览器视频的格式
codecs表示指定视频编码器
